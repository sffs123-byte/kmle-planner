# 오늘 바로 시작 가이드

> 목표: 오늘(토)은 호흡기 1차 루프를 시작하고, 내일(일)은 심장내과 인계/조장 준비를 끝낸다.

---

## 1. 오늘(토) 시작 순서
1. `kmle-planner/index.html` 열기
2. 기존 데이터가 있으면 먼저 **데이터 내보내기**
3. 앱 상단 **`4월 Canary 세팅` 버튼**으로 바로 세팅하기
   - 버튼을 누르면 canary 데이터 적용 후 **오늘 플래너(호흡기 / today / pending)** 로 바로 이동한다
   - 수동으로 할 경우 `data/canary_import_seed.json` **가져오기**
4. 바로 학습으로 들어가려면 **`오늘 학습 시작` 버튼** 사용
   - 아직 canary 세팅 전이면 먼저 canary 세팅을 제안한다
5. `CANARY_TRACKER_APRIL.md`도 같이 열기
6. 호흡기 강의 학습 시작

---

## 2. 오늘(토) 목표
- 호흡기 강의 학습 몰입
- 파트별로 진행 (총론 → 폐기능 장애 → 검사법 같은 식으로 끊어서 보기)
- 최소 몇 파트는 **2단계 직전**까지 끌고 가기
- 밤에 배국자에게 **개념틀 1차 전송**

### 오늘 끝날 때 체크
- [ ] 호흡기 강의 학습 진행
- [ ] 핵심 개념 메모 작성
- [ ] 배국자 1차 전송 완료
- [ ] `CANARY_TRACKER_APRIL.md` 업데이트

---

## 3. 오늘 밤 배국자 전송 예시
```text
호흡기/오늘 본 파트 묶음
- 핵심 개념:
- 감별 포인트:
- 치료/검사 핵심:
- 아직 헷갈리는 점:
- 한줄 정리:
```

### 더 짧은 최소 버전
```text
호흡기/천식-COPD
핵심 감별축 정리했고, ICS 적응증/급성악화 파트가 아직 헷갈림.
```

## 4. 내일(일) 목표
- 심장내과 인계 분석
- 조장 체크리스트 작성
- 첫 3일 준비안 정리

### 내일 사용할 파일

### 내일 끝날 때 체크

- [ ] 조장 업무 분해
- [ ] 과제/발표 확인
- [ ] 첫 3일 준비물 정리
- [ ] 미확정 질문 목록 정리

---

## 5. canary 운영 규칙
- **2단계 완료 = 배국자 개념틀 전송 완료**
- **4단계 완료 = 배국자 함정/출제개념 전송 완료**
- **문풀 total = 알렌 문항 수 데이터(y) 기준**
- 강의/개념틀/오답정리/복습은 파트 수 기준으로 올라간다
- 막히면 `CANARY_TRACKER_APRIL.md`에 **막힘이유**를 남긴다
